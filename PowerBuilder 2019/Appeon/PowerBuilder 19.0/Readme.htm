Introduction

The Appeon PowerBuilder 2019 Maintenance Release 2170 is a setup program that updates "Appeon PowerBuilder" installed from the installation package of Appeon PowerBuilder 2019 Build 2082 (English) or later.

Please note that:

  1. The Universal edition of this Maintenance Release contains the update program for PowerBuilder only, but no update program for PowerServer.
  2. If you have installed PowerBuilder and InfoMaker on the same computer, and if you want to upgrade one of them, you shall install both the PowerBuilder MR and InfoMaker MR, to make sure that the PowerBuilder version is identical with the InfoMaker version.
  3. If you have installed PowerBuilder and InfoMaker on the same computer, and if you want to uninstall one of them, you shall uninstall both, otherwise, the other one will not work properly when one of them is uninstalled.

Checking product versions

This Maintenance Release (hereinafter referred to as MR) can only be applied to Appeon PowerBuilder 2019 Build 2082 or later. The MR setup program will automatically check which products and versions are already installed before performing any update. You can also check the product and version with the following steps:

Step 1: Open Windows Control Panel\Programs\Programs and Features.

Step 2: In the list of currently installed programs, find "Appeon PowerBuilder 2019".

Step 3: Select the product, and view the product version at the bottom of the window. For example, 19.0.0.2082 indicates PowerBuilder 2019 Build 2082.

After MR installation, you can check the product versions using the same instructions, the product version should read 2170 for PowerBuilder.

MR silent installation

You can also run the MR setup program from a command line in the silent mode:

Step 1: Copy the MR installation package to the local disk of the computer.

Step 2: Open a Windows command window, and change to the directory of the MR installation package on the local computer.

Step 3: Execute the silent installer: SilentInstall.bat.

<<TOP

MR uninstallation note

There is no separate uninstall program for PowerBuilder MR. Uninstalling the product will automatically uninstall the MR.
