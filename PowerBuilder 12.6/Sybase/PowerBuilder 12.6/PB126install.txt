PowerBuilder[R] Version 12.6 Installation Notes
*************************************************************************
Before installing PowerBuilder, please read the Installation Guide on the
SAP Help Portal:

   1. Go to the SAP Help Portal at 
        http://help.sap.com/powerbuilder.
   2. Select PowerBuilder 12.6.
   3. Select the Installation Guide.

The Installation Guide contains important information about hardware 
and software prerequisites and detailed information about installing 
the product set. 

To use the demo database, code examples, or tutorial targets, you 
must install SQL Anywhere 16.0 before you install PowerBuilder. 
These features are not installed if the setup program does not 
detect the presence of SQL Anywhere.

You should also read the Release Bulletin for known issues and last 
minute release information. The Release Bulletin is available on the 
SAP Help Portal.
