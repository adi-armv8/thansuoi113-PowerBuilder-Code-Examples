1. Introduction

1.1 Welcome to Appeon PowerBuilder 2018 Beta

Welcome to Appeon PowerBuilder!

Appeon PowerBuilder is an enterprise development tool that allows you to build data-driven business applications and components. It is one of a suite of Appeon products that together provide the tools to develop client/server, Web, and mobile applications.

1.2 Components of PowerBuilder

Different PowerBuilder 2018 Beta editions consist of different components.

Component Universal Edition Standard/Cloud Edition
PowerBuilder IDE Yes Yes

PowerServer (PB Edition)
PowerServer Toolkit
PowerServer Help
PowerServer Web Component

 Yes No
  • PowerBuilder IDE - The development platform for rapidly building distributed, Web, mobile, and client/server applications.
  • PowerServer (PB Edition) - Provides mobile and Web capabilities for the PowerBuilder applications. Not available in the Standalone or Cloud Edition.
    • PowerServer (PB Edition) - A set of server components that are installed to the Microsoft IIS application server and provide the mobile apps and the Web apps with runtime and management services such as data connectivity, DataWindow support, transaction management, and security.
    • PowerServer Toolkit - A set of tools that are installed to the PowerBuilder IDE and deploy PowerBuilder apps to the PowerServer as Web or mobile apps.
    • PowerServer Help - A compiled HTML Help that provides easy use to all user documents.
    • PowerServer Web Component - A series of C++ and JavaScript libraries that are installed to Apache or Microsoft IIS Web server. It is only necessary when configuring a cluster of PowerServers.

3. Installing PowerBuilder 2018 Beta

If you plan to install PowerBuilder and InfoMaker on the same computer, make sure the PowerBuilder version is identical with the InfoMaker version.

If you are running the Web installer of the Beta version, you can follow the online instructions if you choose to install while downloading the product, or follow the instructions below if you choose to install after downloading the product.

Step 1: Double click setup.exe in the installation package.

Step 2: On the welcome page, click Next to proceed.

Step 3: On the License Agreement page, accept the terms in the license agreement page and click Next.

Step 4: On the Customer Information page, enter your name and your company’s name and click Next.

Step 5: On the Choose Destination Location page, click Next to accept the destination path shown, or click Browse and select a different path, click OK, and click Next.

Step 6: On the Choose Destination Location for Shared Files page, click Next to accept the destination path shown, or click Browse and select a different path, click OK, and click Next.

Step 7: On the Specify the SQL Anywhere Engine page, select Yes and then specify the location of a SQL Anywhere engine that will be used to run the demo database (PB demo DB) or select No if no SQL Anywhere engine, and click Next.

Step 8: On the Select Components page, unselect any components you do not want to install, and click Next.

Step 9: On the Select Program Folder page, select the program folder to which program icons will be added, or specify a new one; then click Next.

Step 10: On the Start Copying Files page, review your settings, and then click Next to begin installing files for PowerBuilder.

Step 11: When the installation is complete, the System Reboot Required screen is displayed. Select the Yes, I want to restart my computer now checkbox to reboot the system and then click Finish.

4. Activating PowerBuilder 2018 Beta

You are required to log in to (and activate) PowerBuilder IDE first before using it. If you do not have a valid account to log in, please contact your license administrator to create an account for you. And depending on whether the machine where PowerBuilder is installed has Internet access or not, the account should be assigned with an online license or offline license by your license administrator.

For how to log in and activate Appeon PowerBuilder 2018 Beta, refer to the "Activating PowerBuilder" section in the Appeon License User Guide.

<<Top
 

5. Uninstalling PowerBuilder 2018 Beta

Before you uninstall PowerBuilder, make sure you have logged out PowerBuilder if using an online license, or have deactivated the offline license. For how to log out PowerBuilder or deactivate an offline license, refer to Logging out or De-activating an offline license in the Appeon License User Guide.

Launch Uninstall a program in the Windows Control Panel. Select Appeon PowerBuilder 2018 Beta and click the Uninstall button to begin the uninstall process.
