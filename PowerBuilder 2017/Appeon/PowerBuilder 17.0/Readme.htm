1. Introduction

1.1 Welcome to PowerBuilder 2017

Welcome to PowerBuilder!

PowerBuilder is an enterprise development tool that allows you to build data-driven business applications and components. It is one of a group of Appeon products that together provide the tools to develop client/server, Web, and mobile applications.

1.2 Components of PowerBuilder

Different PowerBuilder 2017 editions consist of different components.

Component Universal Edition Standard/Cloud Edition
PowerBuilder Yes Yes
PowerServer Toolkit Yes No
PowerServer Mobile (.NET) Yes No
  • PowerBuilder - The development tool for rapidly building a client/server application.
  • PowerServer Toolkit - A set of tools that enable the mobile application generation, maintenance, and upgrades within the PowerBuilder IDE.
  • PowerServer Mobile (.NET) - A set of server components that are deployed to the .NET IIS application server and that provide the mobile apps with run-time services such as data connectivity, DataWindow support, transaction management, and security.

1.3 Features of PowerBuilder

Different PowerBuilder 2017 editions contain different features.

Features Universal Edition Cloud Edition Standard Edition
Desktop client/server targets Yes Yes Yes
Mobile targets Yes No No
.NET Web service targets Yes Yes No
.NET Assemblies targets Yes Yes No

2. System Requirements

Before installing PowerBuilder 2017 Universal Edition, please ensure that your system meets the requirements below:

For installing PowerBuilder & PowerServer Toolkit

  • Windows 10, Windows 8.1, Windows 7, or Windows Vista SP2
    Supports Windows Server 2008/2012/2016 runtime only, not development.
  • Microsoft .NET Framework 4.0
  • (Optional) SAP SQL Anywhere 11.0, 12.0, 16.0, or 17.0 (for running the PowerBuilder demos)

For installing PowerServer Mobile (.NET)

  • (Recommended) Windows Server 2016, 2012 R2, or 2008 R2
    (Optional) Windows 10/8.1/7/Vista SP2
  • Microsoft IIS 10, 8.0, 7.5, 7.0, or 6.0
  • Microsoft .NET Framework 4.x

3. Installing PowerBuilder 2017

Step 1: Double click Autorun.exe in the universal edition installation package.

Step 2: Click the Install button on the start page.

Step 3: Click Next to proceed.

Step 4: On the License Agreement page, accept the terms in the license agreement page and click Next.

Step 5: On the Customer Information page, enter your name and your company’s name and click Next.

Step 6: On the Choose Destination Location page, click Next to accept the destination path shown, or click Browse and select a different path, click OK, and click Next.

Step 7: On the Destination Location for Shared Files page, click Next to accept the destination path shown, or click Browse and select a different path, click OK, and click Next. Use the same folder for shared files for all products in this set.

Step 8: On the Specify the SQL Anywhere Engine page, select the location of a SQL Anywhere engine for running the demo database, and click Next.

Step 9: On the Select Products page, select PowerBuilder, PowerServer Mobile, PowerServer Toolkit, and PowerServer Help, and click Next to continue.

The PowerBuilder installation starts first.

  1. In the Select Components page, unselect any components you do not want to install, and click Next.
  2. On the Select Program Folder page, select the program folder to which program icons will be added, or specify a new one; then click Next.
  3. On the Start Copying Files page, review your settings, then click Next to begin installing files for PowerBuilder.
    4.

The PowerServer Mobile installation starts automatically when the PowerBuilder installation is completed.

  1. Click Next until you reach the screen for specifying the IIS Web site where PowerServer Mobile will be installed. Make sure Select an existing Web Site and Default Web Site are selected.
  2. Click Next. Choose whether to install the Demo Applications. If you have SQL Anywhere database server installed, select to install the demo; otherwise, do not install the demo since the demo requires SQL Anywhere database server to be previously installed.
  3. If you selected to install the Demo Applications, specify the path for the SQL Anywhere database server engine.
  4. Click Next until Setup begins installing files for PowerServer Mobile.
    5.

The PowerServer Toolkit installation starts automatically when the PowerServer Mobile installation is completed. Click Next until Setup begins installing files for PowerServer Toolkit.

The PowerServer Help installation starts automatically when the PowerServer Toolkit installation is completed. Click Next until Setup begins installing files for PowerServer Help.

Step 10: When the PowerServer Mobile, PowerServer Toolkit, and PowerServer Help installations are completed, select the checkbox for Restart IIS Server and Register .NET Framework 4.0 and then click Finish.

Step 11: After closing all the windows during installation, the System Reboot Required screen is displayed. Select the Yes, I want to restart my computer now checkbox to reboot the system and then click Finish.

4. Activating PowerBuilder 2017

You are required to log in to (or activate) PowerBuilder IDE first before using it. If you do not have a valid account to log in, please contact your license administrator to create an account for you. And depending on whether the machine where PowerBuilder is installed has Internet access or not, the account should be assigned with an online license or offline license by your license administrator.

For how to log in or activate PowerBuilder, refer to the "Activating PowerBuilder" in the Installation Guide.

5. Uninstalling PowerBuilder 2017

Before you uninstall PowerBuilder, make sure you have logged out PowerBuilder if using an online license or have deactivated the offline license. For how to log out PowerBuilder or deactivate an offline license, refer to "Log out" or "De-activating an offline license" sections in the Installation Guide.

Launch Uninstall a program in the Windows Control Panel. Select Appeon PowerBuilder 2017 and click the Uninstall button to begin the uninstall process.

Take the same steps to uninstall PowerServer Mobile, PowerServer Toolkit, and PowerServer Help.
